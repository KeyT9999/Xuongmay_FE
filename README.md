# Quản Lý Xưởng May - Frontend

Frontend application for the Quản Lý Xưởng May (Garment Factory Management) system.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the app:
   ```bash
   npm run dev
   ```

The app will be available at `http://localhost:5173`
